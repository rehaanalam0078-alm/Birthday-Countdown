🎂 Birthday Countdown App
A simple and elegant Android application that helps you track the countdown to your (or someone special’s) birthday 🎉

📱 Features

⏳ Live countdown timer (days, hours, minutes, seconds)
🎯 Set any custom birthday date
🎨 Clean and minimal UI
⚡ Lightweight and fast performance
📆 Automatically updates countdown in real-time


🚀 Tech Stack

Language: Kotlin
IDE: Android Studio
UI: XML Layouts
Platform: Android SDK

App calculates the remaining time
Countdown updates every second dynamically
